Takes documentation from GitHub to populate the 360Giving website.

== Description ==
Much of the documentation for the 360Giving Data Standard is maintained in
a single source of truth on GitHub: https://github.com/ThreeSixtyGiving/standard/

There are also a number of scripts in that repository that generate various
templates and so on.

This plugin enables the website to use the content in that repository.


== Installation ==
1. Upload "threesixty_docs" directory to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Follow the infomation at https://github.com/ThreeSixtyGiving/website#how-do-the-docs-work
